🚀 Employee Management System (EMS)

A full-stack Employee Management System built using JSP, Servlets, HTML, CSS, and MySQL to manage employees, departments, attendance, payroll, and performance efficiently.

📌 Project Status:-

🛠️ Under Development – Core modules are implemented, and advanced features are being added.

🎯 Features:-

👨‍💼 Admin Panel:-

Admin Login
Manage Managers & Employees (Add / Update / Delete)
Assign Roles & Permissions
Department Management
Salary Structure Setup
View Reports & Analytics
System Settings (Configurations & Access Control)

👨‍💻 Manager (HR) Panel:-

Manager Login
Add New Employees
Attendance Management
Leave Approval / Rejection
Performance Reviews
Payroll Processing
Document Upload
Generate Reports (PDF)

🔮 Future Scope (Employee Panel):-

Employee Login System
View & Update Profile
Apply for Leave
Track Attendance


🛠️ Tech Stack:-

Frontend: HTML, CSS, JavaScript
Backend: JSP & Servlets (Java)
Database: MySQL (XAMPP)
Server: Apache Tomcat


⚙️ Setup Instructions:-

Clone the repository
(git clone https://github.com/your-username/EmployeeManagementSystem.git)
Import project into Eclipse / IntelliJ
Configure Apache Tomcat Server
Setup MySQL Database via XAMPP
Update database credentials in:
db/db.jsp
Run the project on server
